Ejercicio AJAX

Hola! soy el archivo 1

  1. Apenas abierta la página, se deberá solicitar el archivo archivo-1.txt y mostrar su contenido en la consola cuando el mismo se reciba.
  2. Adicionar un botón que al ser presionado inicie el mismo procedimiento anterior.
  3. Adicionar otro botón que, al ser presionado, inicie una petición al servidor, solicitando el archivo-2.txt. Al recibirse el contenido, deberá mostrarse dentro del container .container
  4. Bonus 1:
    1. Parte A: Adicionar otro botón que, al ser presionado, inicie una petición al servidor, solicitando el archivo-2.txt. Al recibirse el contenido, no deberá mostrarse el mismo, sino que deberá aparecer un nuevo botón con el texto "Mostrar contenido". Dicho botón, al momento de ser presionado, deberá mostrar el texto "Botón presionado".
    2. Parte B: Si funciona lo anterior, lograr que este nuevo botón generado muestre el contenido recibido previamente.
  5. Bonus 2: Adicionar otro botón, que reciba el contenido del archivo archivo-1.txt y que, cada vez que sea presionado, agregue el contenido de dicho archivo a un nuevo párrafo ubicado dentro del contenedor container.
